# Controle de Estoque

## Descrição

Este projeto é um sistema simples de controle de estoque desenvolvido para a disciplina de Análise e Desenvolvimento de Sistemas. Permite cadastrar produtos com nome, quantidade e preço, além de editar e excluir registros. Também exibe um resumo com o total de itens e o valor total do estoque.

---

## Funcionalidades

- Adicionar produtos com nome, quantidade e preço unitário  
- Editar produtos cadastrados  
- Excluir produtos da lista  
- Visualizar o total de itens cadastrados  
- Visualizar o valor total do estoque  

---

## Tecnologias Utilizadas

- **HTML5**: Estrutura da aplicação  
- **CSS3**: Estilos visuais e layout  
- **JavaScript (ES6)**: Lógica e interatividade  

---

## Como usar

1. Clone ou baixe o repositório  
2. Abra o arquivo `index.html` em um navegador moderno  
3. Preencha o formulário para adicionar produtos  
4. Use os botões de **Editar** e **Excluir** para modificar ou remover produtos  
5. Acompanhe o resumo do estoque na parte inferior da página  

---

## Estrutura dos arquivos

- `index.html` — Estrutura principal do projeto  
- `style.css` — Estilos do projeto  
- `script.js` — Código JavaScript para funcionalidades  

---

## Observações

Este projeto foi desenvolvido para fins educacionais e não utiliza banco de dados, portanto os dados são armazenados apenas enquanto a página está aberta (em memória).
